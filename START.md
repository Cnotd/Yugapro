## 📋 使用步骤

1. **打开浏览器** 访问 http://localhost:5000

2. **上传瑜伽视频**
   - 点击"选择文件"
   - 支持格式：mp4, avi, mov
   - 文件大小：≤100MB
   - 视频时长：≤2分钟

3. **选择动作类型**
   - 下犬式
   - 树式
   - 战士一式
   - 三角式
   - 猫牛式

4. **点击"开始评估"**
   - 系统自动分析视频
   - 检测人体关键点
   - 计算关节角度
   - 调用AI模型评估

5. **查看结果**
   - 总分 (0-100分)
   - 详细评分 (角度准确性、稳定性、协调性)
   - 主要问题列表
   - 改进建议
   - 可下载标注视频

## 🎯 评估维度

- **角度准确性** (40分): 主要关节角度是否在标准范围内
- **动作稳定性** (30分): 动作是否稳定，是否有过大晃动
- **整体协调性** (30分): 动作是否流畅，身体各部位是否协调

## 💡 使用建议

### 视频拍摄技巧
1. **光线充足**：确保环境光线明亮
2. **背景简洁**：选择干净的背景
3. **完整动作**：拍摄完整的瑜伽动作
4. **正面或侧面**：选择最佳角度拍摄
5. **人物清晰**：确保人物在画面中清晰可见

### 获得准确评估
1. 拍摄15-30秒的视频即可
2. 选择光线明亮的环境
3. 避免遮挡和反光
4. 保持动作稳定
5. 可以多次上传不同角度的视频

## 🔧 系统配置

### 当前配置
- **大模型**: Ollama qwen:4b
- **姿态检测**: MediaPipe Pose
- **视频处理**: OpenCV
- **Web服务**: Flask (端口 5000)

### 更换模型
如需更换 Ollama 模型，编辑 `config/settings.py`:

```python
OLLAMA_CONFIG = {
    "host": "http://localhost:11434",
    "model": "qwen:4b",  # 修改这里
    "timeout": 30,
    "temperature": 0.7,
}
```

### 安装其他 Ollama 模型

```bash
# 安装 Qwen 7B (支持视觉)
ollama pull qwen:7b

# 安装 Llama3
ollama pull llama3:8b

# 安装其他模型
ollama list  # 查看所有可用模型
```

## 🆘 常见问题

### Q: 评估速度慢？
A: 首次运行会加载模型，后续会更快。视频越长处理时间越久。

### Q: 关键点检测失败？
A: 确保视频中有清晰可见的人物，光线充足。

### Q: AI评估不可用？
A: 检查 Ollama 是否运行，模型是否已下载。

### Q: 如何提高评估准确性？
A: 使用高质量视频，从最佳角度拍摄，确保动作标准。
