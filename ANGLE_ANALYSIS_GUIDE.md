# 角度检测和标注系统使用指南

## 功能概述

基于已标注的关键点信息，进行关节角度计算、可视化和评估。

## 输出内容

处理视频后，会在归档目录下创建 `angle_analysis` 文件夹：

```
{归档目录}/{视频名}/angle_analysis/
├── best_frames_with_angles/    # 带角度标注的最佳帧
│   ├── frame_1_angles.jpg
│   ├── frame_2_angles.jpg
│   └── frame_3_angles.jpg
├── angle_summary.png            # 角度汇总表格
├── angle_time_series.png        # 角度时间序列图表
├── angle_data.json             # 完整角度数据
└── angle_assessment_report.txt  # 评估报告
```

## 使用方法

### 1. 处理单个视频

```bash
python temp_tests/test_angle_single.py
```

**功能**：
- 从归档视频采样帧
- 计算每个帧的所有关节角度
- 生成角度标注的最佳帧
- 创建分析图表和报告

**参数**：
- 采样率：每 2 秒 1 帧
- 质量过滤：置信度 > 0.5，有效点比例 > 0.7
- 输出：3 张最佳帧 + 分析报告

### 2. 批量处理

```bash
python temp_tests/test_angle_analysis.py
```

**功能**：
- 处理归档目录下的所有视频
- 为每个视频生成角度分析
- 支持选择特定动作

## 计算的角度

系统计算以下 10 个角度：

| 角度名称 | 说明 | 关键点 |
|---------|------|---------|
| left_elbow | 左肘角度 | 左肩-左肘-左腕 |
| right_elbow | 右肘角度 | 右肩-右肘-右腕 |
| left_knee | 左膝角度 | 左髋-左膝-左踝 |
| right_knee | 右膝角度 | 右髋-右膝-右踝 |
| left_hip | 左髋角度 | 左肩-左髋-左膝 |
| right_hip | 右髋角度 | 右肩-右髋-右膝 |
| left_shoulder | 左肩角度 | 左肘-左肩-左髋 |
| right_shoulder | 右肩角度 | 右肘-右肩-右髋 |
| spine_angle | 脊柱角度 | 肩膀中点到髋部中点 |
| body_tilt | 身体倾斜角 | 左右肩连线角度 |

## 评估指标

### 1. 平均角度

每个关节在整个视频中的平均角度值。

**应用**：
- 评估标准姿势
- 比较不同动作的差异
- 识别异常角度

### 2. 角度稳定性（标准差）

衡量关节角度随时间的变化程度。

**判定标准**：
- 标准差 < 5°：稳定 ✓
- 标准差 ≥ 5°：不稳定 ✗

**应用**：
- 评估动作稳定性
- 识别抖动或调整动作
- 评估整体动作质量

### 3. 整体稳定性

稳定关节数 / 总关节数

**评价等级**：
- ≥ 80%：动作非常稳定
- 50%-80%：动作比较稳定
- < 50%：动作不够稳定，需要改进

## 输出文件说明

### 1. 带角度标注的最佳帧

显示人体关键点和关节角度标注：

- **绿色圆点**：关键点位置
- **黄色弧线**：角度指示
- **黄色文字**：角度名称和数值
- **黄色连线**：关节连接线

**用途**：
- 直观查看关节角度
- 评估标准姿势
- 对比不同帧

### 2. 角度汇总表格 (`angle_summary.png`)

显示所有关节的平均角度和稳定性：

- **Joint**：关节名称
- **Avg Angle**：平均角度
- **Std Dev**：标准差
- **Status**：稳定性状态（绿色=稳定，红色=不稳定）

**用途**：
- 快速概览所有关节
- 对比左右对称性
- 识别问题关节

### 3. 角度时间序列图表 (`angle_time_series.png`)

包含 4 个子图：

1. **肘关节角度变化**
   - 左肘 vs 右肘
   - 时间轴显示角度变化

2. **膝关节角度变化**
   - 左膝 vs 右膝
   - 时间轴显示角度变化

3. **髋关节角度变化**
   - 左髋 vs 右髋
   - 时间轴显示角度变化

4. **关节角度稳定性**
   - 各关节的标准差柱状图
   - 对比不同关节的稳定性

**用途**：
- 分析动作动态变化
- 识别关键时间点
- 评估动作流畅度

### 4. 角度数据 (`angle_data.json`)

JSON 格式的完整角度数据：

```json
{
  "video_name": "视频名称",
  "total_frames": 帧数,
  "avg_angles": {
    "left_elbow": 169.9,
    "right_elbow": 154.2,
    ...
  },
  "std_angles": {
    "left_elbow": 8.73,
    "right_elbow": 13.49,
    ...
  },
  "angle_sequence": [
    {
      "timestamp": 0.0,
      "angles": {
        "left_elbow": 169.9,
        ...
      }
    },
    ...
  ]
}
```

**用途**：
- 程序化分析
- 数据导出
- 二次分析

### 5. 评估报告 (`angle_assessment_report.txt`)

文本格式的详细评估报告：

```
============================================================
角度分析评估报告
============================================================

视频名称: video_annotated
处理帧数: 3
采样率: 0.5 帧/秒

1. 平均角度
----------------------------------------
  left_elbow: 169.9°
  right_elbow: 154.2°
  ...

2. 角度稳定性（标准差）
----------------------------------------
  left_elbow: 8.73° (不稳定)
  right_elbow: 13.49° (不稳定)
  ...

3. 整体评估
----------------------------------------
  稳定关节数: 3/10
  整体稳定性: 30.0%
  评价: 动作不够稳定，需要改进
```

**用途**：
- 快速阅读评估结果
- 存档记录
- 分享给用户

## 标注说明

### 颜色含义

- **绿色圆点**：关键点位置
- **黄色弧线**：角度指示弧线
- **黄色连线**：关节连接线
- **黑色背景文字**：角度标注

### 角度命名

角度标注显示格式：`关节名: 角度值°`

例如：
- `左肘: 169.9°`
- `右膝: 168.3°`

## 实际应用

### 应用1：动作标准性评估

**方法**：对比平均角度与标准值

**示例**：
- 标准下犬式：左膝 170°
- 实测左膝：168.3°
- 评估：符合标准

### 应用2：动作稳定性评估

**方法**：查看角度标准差

**示例**：
- 标准差 < 5°：稳定 ✓
- 标准差 ≥ 5°：需要改进 ✗

### 应用3：左右对称性评估

**方法**：对比左右关节的平均角度

**示例**：
- 左肘：169.9°
- 右肘：154.2°
- 差异：15.7°（不对称）

### 应用4：动作改进建议

**方法**：基于不稳定的关节给出建议

**示例**：
- 肘关节不稳定：建议加强手臂控制
- 肩关节不稳定：建议放松肩膀

## 注意事项

1. **采样率**：默认每 2 秒 1 帧，可在代码中调整
2. **质量过滤**：只分析通过质量过滤的帧
3. **角度范围**：0-180 度，使用最小内角
4. **稳定性阈值**：标准差 < 5° 为稳定
